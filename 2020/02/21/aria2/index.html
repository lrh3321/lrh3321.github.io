使用 aria2 搭建离线下载服务器

在VPS主机上安装 aria2 下载软件用作离线下载，能够远程管理下载，下载完成后能够通过HTTP（或其他任何你愿意的方式）从VPS取回。

安装

Ubuntu/Debian 系统

其他操作系统

可以访问Aria2 官网获取下载、安装信息。

配置&启动

aria2 最简单的运行方式是 aria2c <download-url-path>，即会从给出的url下载文件。但我们希望它能记住一些设置，常驻后台运行。aria2 启动时会首先尝试从$HOME/.aria2/aria2.conf读取配置文件。此外，可以在启动时加入--conf-path参数手动指定配置文件位置，也可以通过--no-conf强制不读取配置文件。

配置文件取回功能

我们希望 aria2 下载好文件后，我们能够通过浏览器访问下载目录，并能下载文件。可以搭建 HTTP 或 FTP 服务来实现这一功能。通常 HTTP 和 FTP 搭建的文件服务，可以使用下载工具，一些服务软件（如 nginx ）还可以支持多线程下载、断点续传。
